# Restaurant Project

Live Demo: [Restaurant Project](https://restaurant-mu-bay.vercel.app/)

## Project Overview

This project is a **restaurant website** built with Next.js. It features:

- Responsive design for mobile, tablet, and desktop.
- Dynamic categories and food items.
- Interactive UI components with **Framer Motion** animations.
- Image optimization using **Next.js Image** component.
- Client testimonials slider with **React Slick**.
- Modal functionality for adding new food items and categories.
- Tailwind CSS for styling.
- React Query for fetching and caching data.
- Fully functional and easy to extend for additional features.

## Getting Started

First, run the development server:

```bash
npm run dev

```
